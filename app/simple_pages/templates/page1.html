Git

Git is a free and open-source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. An in-progress, or completed, project's code can be stored in a GitHub repository for proper management and version-control of the source code. Git is regarded as the dominant source code management software (SCM) at this time.
